#Gerenciador de Tarefas Pro
Este é um projeto de Lista de Tarefas (To-Do List) focado em organização pessoal, desenvolvido para meu portfólio.

Funcionalidades
* Adição de tarefas com título, descrição e data prazo.
* Upload e visualização de arquivos anexos.
* Sistema de busca em tempo real.
* Histórico de atividades concluídas.
* Barra de progresso dinâmica.
* Layout 100% Responsivo (Mobile First).

Tecnologias
* HTML5
* CSS3 (Flexbox e Media Queries)
* JavaScript (Manipulação de DOM e LocalStorage)
